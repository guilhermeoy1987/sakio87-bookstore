# sakio87-bookstore

🧪 Testes Automatizados e Qualidade de Software (Bookstore API)
Data: 2026 | Foco: Test-Driven Development (TDD) e Estabilidade
Desenvolvimento focado na branch novoteste, implementando uma camada de testes automatizados para garantir que a lógica de negócio e os endpoints da API funcionem exatamente como esperado, prevenindo bugs em futuras atualizações.

✅ Testes de Integração: Implementação de scripts para testar o fluxo completo da API, desde a recepção da requisição até a resposta final, simulando o uso real do sistema.

✅ Validação de Endpoints: Uso do APITestCase do Django Rest Framework para verificar códigos de status HTTP (200 OK, 201 Created, 400 Bad Request) e a integridade dos dados retornados.

✅ Garantia de Regressão: Criação de uma suíte de testes que permite refatorar o código com segurança, garantindo que novas funcionalidades não quebrem o que já está funcionando.

✅ Automação de Cenários: Testes específicos para criação, listagem e busca de livros, validando se as regras de negócio (como campos obrigatórios e tipos de dados) estão sendo respeitadas.

Stack: Python, Django Rest Framework (Testing), Pytest/Unittest e Git.
