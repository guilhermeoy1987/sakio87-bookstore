🔐 Autenticação e Segurança em APIs (Bookstore API)
Data: 2026 | Foco: Segurança, Controle de Acesso e Proteção de Dados
Desenvolvimento focado na branch autenticacao, elevando o nível de maturidade da API ao implementar camadas de segurança que garantem que apenas usuários autorizados possam acessar ou modificar informações sensíveis.

✅ Implementação de Autenticação: Configuração de sistemas de login e identificação de usuários, utilizando padrões de mercado para validar a identidade de quem acessa a API.

✅ Controle de Permissões (ACL): Definição de níveis de acesso (ex: Administrador vs. Cliente), garantindo que operações críticas como deletar ou editar livros sejam restritas a perfis específicos.

✅ Segurança de Endpoints: Proteção das rotas da API contra acessos anônimos, assegurando a integridade dos dados e prevenindo vulnerabilidades comuns de segurança.

✅ Serialização Segura: Ajuste dos Serializers para garantir que informações sensíveis de usuários não sejam expostas nas respostas JSON da API.

Stack: Python, Django, Django Rest Framework (Auth), Git.
